LidBot is a Telegram userbot that connects to specified channels, analyzes incoming messages, and forwards only relevant client requests. The project is designed for niches that require automation of lead generation and message filtering.

Features

Authorization as a user through Telethon

Monitoring of channel and chat lists

Multi-level message filtering:

advertising and agent posts

irrelevant topics

Welcome and selling posts

Messages with phone numbers or URLs

Forwarding only targeted leads to the specified chat or bot

Maintaining a cache of channel IDs

Logging of skips and filtering reasons

Technologies

Python 3.10+

Telethon

Python-dotenv

Structured logic for filters, parsing, and notifications

Application

It is suitable for automating processes in the fields of car rental, services, sales, logistics, request aggregators, and any tasks that require tracking customer inquiries in Telegram.
